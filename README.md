# packaging

## Windows
```pwsh
python.exe -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python.exe setup.py sdist
```

## Linux

```sh
python3 -m venv venvl
source ./venvl/bin/activate
pip install -r requirements.txt
python3 setup.py sdist
```
